Lektion 12: Daten ändern und löschen (UPDATE & DELETE)

Was ist UPDATE?

Mit UPDATE kannst du vorhandene Daten in einer Tabelle ändern. Wichtig ist die WHERE-Klausel, damit nicht alle Datensätze verändert werden.

UPDATE Kunde
SET Ort = 'Hamburg'
WHERE Name = 'Bernd';

Was ist DELETE?

DELETE entfernt komplette Zeilen aus einer Tabelle. Auch hier ist WHERE sehr wichtig, um gezielt zu löschen:

DELETE FROM Bestellung
WHERE Betrag < 100;

Beispieltabelle: Kunde

+----+--------+--------+
| ID | Name   | Ort    |
+----+--------+--------+
| 1  | Anna   | Köln   |
| 2  | Bernd  | Berlin |
| 3  | Clara  | Köln   |
| 4  | David  | Köln   |
+----+--------+--------+

Aufgabe 1

Ändere den Ort von „Clara“ auf „München“.

Aufgabe 2

Lösche alle Kunden mit Ort „Köln“.
