Einführung: Eine Datenbank Schritt für Schritt aufsetzen

In dieser Lektion lernst du Schritt für Schritt, wie du eine SQL-Datenbank aufsetzt und grundlegende Befehle anwendest:

  • Datenbank erstellen
  • Tabellen anlegen
  • Daten einfügen
  • Daten gezielt abfragen
  • Daten aktualisieren
  • Daten löschen

Kurzes Video zum Einstieg:

📂 Datenbank erstellen

Mit dem Befehl CREATE DATABASE legst du eine neue Datenbank an.

Beispiel:

CREATE DATABASE Werkstatt;

✏️ Aufgabe:

Erstelle eine Datenbank mit dem Namen Autohaus.

📑 Tabelle anlegen

Mit dem Befehl CREATE TABLE legst du Tabellen in deiner Datenbank an.

Beispiel:

Diese Tabelle speichert Kundendaten:

CREATE TABLE Kunden (
  ID INT PRIMARY KEY,
  Vorname VARCHAR(50),
  Nachname VARCHAR(50),
  Email VARCHAR(100)
);

✏️ Aufgabe:

Erstelle eine Tabelle Fahrzeuge mit folgenden Spalten:

  • ID (INT)
  • Marke (VARCHAR(50))
  • Modell (VARCHAR(50))
  • Preis (INT)

➕ Daten einfügen

Mit dem Befehl INSERT INTO kannst du neue Datensätze in eine Tabelle einfügen.

Beispiel:

Füge einen Kunden zur Tabelle Kunden hinzu:

INSERT INTO Kunden (ID, Vorname, Nachname, Email)
VALUES (1, 'Max', 'Muster', 'max@example.com');

✏️ Aufgabe:

Füge folgendes Fahrzeug in die Tabelle Fahrzeuge ein:

  • ID: 1
  • Marke: 'Audi'
  • Modell: 'A3'
  • Preis: 23990

🔍 Daten gezielt abfragen

Mit dem Befehl SELECT kannst du Daten aus einer Tabelle auslesen.

Beispiel:

Alle Fahrzeuge anzeigen:

SELECT * FROM Fahrzeuge;

Nur Marke und Modell anzeigen:

SELECT Marke, Modell FROM Fahrzeuge;

✏️ Aufgabe:

Zeige alle Daten vom Fahrzeug mit der ID 1 an.
